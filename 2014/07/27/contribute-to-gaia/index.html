So you've heard about this Firefox OS and you knew enough about it that you are new very encouraged to contribute code to its User Interface layer called "Gaia". Of course by now you know that Gaia is completely written in modern web technologies such as HTML5, CSS3 and of course JavaScript.
In this blog post, I'm going to show you, based on my experience, everything about contributing code to Gaia and having it successfully merged and landed into Firefox OS' code base.
